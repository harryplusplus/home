## Overview

Github: [link](https://github.com/harryplusplus/kero-variant-utils)

- Ergonomic and Header-Only: A C++ library designed for simpler `std::variant` alternative handling.
- Compile-Time Validation: Ensures all `std::variant` alternatives are safely handled, improving code robustness.
- Wide Compatibility: Supports C++17, C++20, and C++23 standards, and major compilers like GCC, Clang, and MSVC.
- Easy Integration: Can be integrated via CMake FetchContent, Installable CMake Package, or Manual Installation.
